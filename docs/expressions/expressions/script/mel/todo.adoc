* Gotchas/FAQ

`Field selections must be performed on messages or maps.`
This means that the value may be null.
E.g. for code `input.orig`, variable `input` may be null.
Solution: `input.?orig`

`found no matching overload for '_?_:_' applied to '(bool, optional_type(string), string)' (candidates: (bool, %A8, %A8))`


CEL is evaluating *both* options in conditional (`?:`) before making a decision.
